\startEXERCISE
有向圖 $G=(V,E)$ 的\emph{轉置}是圖 $G^T=(V,E^T)$，
其中 $E^T=\{(v,u)\in V \times V:(u,v)\in E\}$。
即 $G^T$ 是將 $G$ 中所有邊反向。
設計一個高效算法由 $G$ 計算 $G^T$，
用鄰接鏈表和鄰接矩陣兩種方法表示 $G$。
並分析算法的複雜度。
\stopEXERCISE

\startANSWER
鄰接鏈表表示法：$O(V+E)$，需要額外大小爲 $O(V)$ 的指針數列。

\CLRSH{TRASPOSE(A)}
\startCLRSCODE
A = Adj[1 : |V|]
A' = \ALGO{NEW(Adj'[1 : |V|])}
for u = 1 to |V|
	for \text{顶点} v in G.Adj[u]
		\ALGO{INSERT(Adj'[v], u)}
return A'
\stopCLRSCODE

鄰接矩陣表示法：轉置矩陣即可， $O(|V|^2)$。

\CLRSH{TRASPOSE(A)}
\startCLRSCODE
A = |V| \times |V|
A' = \ALGO{NewMatrix(|V| \times |V|)}
for i = 1 to |V|
	for j = 1 to |V|
		A'[j][i] = A[i][j]
return A'
\stopCLRSCODE

\stopANSWER
